## 📌 Descripción

**RutaRD** es un prototipo web que busca solucionar la falta de información del transporte público en la República Dominicana, ofreciendo una plataforma visual donde los usuarios pueden consultar rutas, conocer recorridos y acceder a información básica de transporte.

Este proyecto está enfocado en la validación de la idea (fase de prototipo), priorizando el diseño, la experiencia de usuario y la estructura del sistema.

## 🎯 Objetivo del Proyecto

Desarrollar un prototipo web que permita simular una plataforma de consulta de transporte público, facilitando el acceso a información clave de manera clara, rápida y accesible.

## 🌍 Impacto Social

Este sistema contribuye a mejorar la movilidad urbana y el acceso a la información, alineándose con iniciativas de desarrollo sostenible impulsadas por la Organización de las Naciones Unidas.

- ✅ Mejora la planificación de rutas
- ✅ Reduce la desinformación
- ✅ Optimiza el tiempo de los usuarios
- ✅ Promueve ciudades más organizadas

## 🧠 Problema Identificado

En la República Dominicana:
- ❌ No existe una plataforma centralizada de rutas.
- ❌ Los horarios son informales.
- ❌ La información se obtiene de manera empírica.
- ❌ El transporte es mayormente informal.

## 💡 Solución Propuesta

Una plataforma web interactiva que permita:
- 🚌 Consultar rutas de transporte (Guaguas, Metro, Carros Públicos).
- 🗺️ Visualizar recorridos paso a paso.
- 💰 Conocer precios estimados y horarios.
- 📱 Simular una experiencia real de uso colaborativo.

## ⚙️ Alcance

### ✔️ Incluye
* Interfaz web completa e interactiva
* Navegación entre múltiples páginas
* Simulación de funcionalidades (filtros, valoraciones, estados)
* Diseño completamente *Responsive*

### ❌ No incluye (En esta fase)
* Backend / API
* Base de datos real
* Autenticación de usuarios
* Datos GPS en tiempo real

## 📁 Estructura del Proyecto

```text
ruta-rd/
│
├── index.html        # Página principal
├── rutas.html        # Listado y filtrado de rutas
├── detalle.html      # Información específica de la ruta
├── sobre.html        # Acerca del proyecto y equipo
│
├── css/
│   └── styles.css    # Estilos personalizados y variables
│
├── js/
│   ├── app.js        # Lógica interactiva
│   └── data.js       # Simulación de base de datos
│
└── libs/             # Bibliotecas locales (Bootstrap)
```

## 🧩 Funcionalidades del Prototipo

- 🔍 Simulación de búsqueda de rutas.
- 🗺️ Visualización de origen/destino y recorridos.
- 📍 Listado de rutas divididas por tipos de transporte.
- 📄 Detalle de cada ruta con precios y horarios aproximados.
- ⭐ Valoraciones visuales dinámicas.
- 💬 Comentarios y reportes de tráfico simulados.

## 🎨 Diseño UI/UX

El diseño está enfocado en:
* Experiencia de usuario intuitiva (Mobile-first).
* Diseño limpio, moderno y atractivo (Glassmorphism sutil).
* Uso de componentes estructurales reutilizables.

**Paleta de colores:**
* 🔵 **Azul:** Elementos principales (Confianza e Institucionalidad).
* 🟢 **Verde/Múltiples:** Acciones y tipos de transporte.
* ⚪ **Gris/Blanco:** Fondos claros para contraste óptimo.

### 📱 Responsive Design
El sistema está optimizado y probado rigurosamente para:
* 📱 **Móviles** (Smartphones)
* 📲 **Tablets**
* 💻 **Escritorio** (Laptops & Desktops)

## 🚀 Instalación y Uso

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/Alanedberg/ruta-rd.git
   ```
2. **Acceder al directorio:**
   ```bash
   cd ruta-rd
   ```
3. **Ejecutar el proyecto:**
   Simplemente abre el archivo `index.html` en tu navegador de preferencia o usa una extensión como *Live Server*.
